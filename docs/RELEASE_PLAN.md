## Reproducibility Targets

The final release is expected to include scripts for:

- success, collision, timeout, and path-efficiency evaluation;
- latency breakdown measurement;
- ADE/FDE diagnostics;
- near-miss and blockage-time metrics;
- induced participant deviation and hard-braking metrics;
- Isaac Sim and ROSOrin trial replay where possible.
